Needle in a Haystack

Scenario

An intelligence dump was recovered from a compromised server. The file contains thousands of log entries, garbage data, and decoys. Somewhere in those 10,000 lines an operative embedded the flag on a line marked with the keyword FLAGLINE.

You could scroll through 10,000 lines manually. Or you could use the right tool — the one every Linux admin reaches for first.

How To Solve

1
Download the file using the button above or via terminal:
wget https://your-repo.github.io/files/data.txt
2
Run grep to search for the keyword FLAGLINE — it instantly returns the one matching line from 10,000.
3
Read the flag — the format is FLAGLINE :: FlagVault{...}. Copy the full flag value and submit below.

Command Examples

bash — basic grep
$ grep "FLAGLINE" data.txt
FLAGLINE :: FlagVault{gr3p_1s_y0ur_fr13nd}
bash — show line number too
$ grep -n "FLAGLINE" data.txt
5853:FLAGLINE :: FlagVault{gr3p_1s_y0ur_fr13nd}
bash — extract flag value only (pipe + cut)
# Pipe into cut to isolate just the flag
$ grep "FLAGLINE" data.txt | cut -d' ' -f3
FlagVault{gr3p_1s_y0ur_fr13nd}
bash — awk alternative
# awk scans for pattern, prints full line
$ awk '/FLAGLINE/' data.txt
FLAGLINE :: FlagVault{gr3p_1s_y0ur_fr13nd}
PowerShell — for Windows users
# On Windows — PowerShell equivalent
PS> Select-String -Pattern "FLAGLINE" -Path data.txt
data.txt:5853:FLAGLINE :: FlagVault{gr3p_1s_y0ur_fr13nd}
python — cross-platform one-liner
$ python3 -c "print([l.strip() for l in open('data.txt') if 'FLAGLINE' in l][0])"
FLAGLINE :: FlagVault{gr3p_1s_y0ur_fr13nd}

Hints

Use grep — Global Regular Expression Print. It reads every line and prints lines that match your pattern.

Syntax: grep "PATTERN" filename
grep "FLAGLINE" data.txt

This searches all 10,000 lines for the word FLAGLINE and immediately returns the single line containing it — plus the flag.
Options for Windows users:

1. Git Bash — ships with grep, free from git-scm.com
2. WSL — Windows Subsystem for Linux
3. PowerShell: Select-String -Pattern "FLAGLINE" data.txt
4. Python: python3 -c "print([l for l in open('data.txt') if 'FLAGLINE' in l][0].strip())"
